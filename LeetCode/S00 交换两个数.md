## 中间变量

```c++
int temp=a;
a=b;
b=temp;
```

* 有额外中间变量空间消耗

## 相加寄存

```c++
a=a+b;
b=a-b;
a=a=b;
```

* 无额外空间消耗
* 当``a+b>INT_MAX``时，此时会产生正溢出，导致最终结果错误

## 按位异或

```c++
b=a^b;
a=a^b;
b=a^b;
```

```c++
s[i] ^= s[j];
s[j] ^= s[i];
s[i] ^= s[j];
```

```c++
//例子
 a=10100001,b=00000110;
 a=a^b; 　　//a=10100111
 b=b^a; 　　//b=10100001
 a=a^b; 　　//a=00000110
```

* 无额外空间消耗
* 原理

| 初始值 | a       | b       |
| ------ | ------- | ------- |
| b=a^b  | a       | a^b     |
| a=a^b  | a^a^b=b | a^b     |
| b=a^b  | b       | a^b^b=a |

* 另外可以认为按位异或是挑某几位翻转
